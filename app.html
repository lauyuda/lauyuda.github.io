NUSmoney App

Created the relational database of users, accounts, and transactions.
Backend retrieved relevant data using SQL to join different tables and
frontend to display data with graphical visualisation.

Backend

Relational Database

Created a database consisting of three tables using random values.

API

Connected to MySQL database and created router objects to define URI mappings to run queries for requests.

Frontend

Tables

Each user and each account is unique in the database. Frontend allows users to key in their ID and view all their different account balance at once. They can also look at past transactions across different accounts too.

Graphs

Bar charts added to help users visualise the proportion of their money in each account they own. Transactions bar chart shows green for a positive cash flow and red for negative cash flow; amount is given by height of bar chart.
